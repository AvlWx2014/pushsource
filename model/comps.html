Push items: comps.xml¶

class pushsource.CompsXmlPushItem[source]¶

A push item representing a comps XML file.

For push items of this type, the src attribute refers to a file containing a comps XML file (package group definitions) in the format used by yum & dnf.

This library does not verify that the referenced file is valid.
